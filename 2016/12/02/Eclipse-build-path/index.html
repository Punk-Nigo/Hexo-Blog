Eclipse build path

Eclipse 導入錯誤提示 Setting build path….

如果再導包的時候出現提示’Setting build path has….’警告視窗!!

請開啟該署專案資料夾，裏頭.classoath 文件刪除即可~

之後再做一次導包就好了喔。
