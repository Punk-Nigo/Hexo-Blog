hexo install

前言…

真的是無聊上網看到別人架站使用hexo，就想說來試試手氣。

有趣的事~一次deploy，可以同時發佈到github heroku firebase。

當然你可以不用像我這樣無聊😬

Hexo Cli 快速上手

安裝

熱鍵 win + R 

輸入 cmd 或 PowerShell

npm install hexo-cli -g

版本

hexo version

hexo: 3.8.0
hexo-cli: 1.1.0

測試

hexo #

建立專案

hexo init Blog

Blog = 資料夾

cd Blog

檢視專案

hexo server 或指定 hexo server -p 8080

預設4000 port

hexo install
